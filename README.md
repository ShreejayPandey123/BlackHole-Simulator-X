<img width="918" height="654" alt="image" src="https://github.com/user-attachments/assets/5b2c7263-7292-4292-985f-f049df65e34a" /># 🌌 Black Hole Simulator X

A real-time Black Hole Simulator developed using **Python**, **Panda3D**, and **GLSL shaders**.

This project aims to visualize a realistic black hole environment inspired by modern astrophysics, featuring an event horizon, glowing accretion disk, photon ring, plasma effects, and a procedurally generated galaxy.

---

## ✨ Features

- 🌑 Realistic Black Hole Event Horizon
- 🔥 Animated Accretion Disk
- 💫 Procedurally Generated Spiral Galaxy
- ⭐ Dynamic Star Field
- ☄️ Plasma Particle Simulation
- 🌟 Bloom Post-Processing
- 🎨 Custom GLSL Vertex & Fragment Shaders
- 🎥 Interactive Camera Controls
- ⚡ Real-Time Rendering with Panda3D

---

## 🛠️ Technologies Used

- Python 3.x
- Panda3D 1.10.16
- OpenGL (GLSL 330)
- NumPy

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/BlackHole-Simulator-X.git
```

2. Install Panda3D:

```bash
pip install panda3d
```

3. Run the simulator:

```bash
python main.py
```
